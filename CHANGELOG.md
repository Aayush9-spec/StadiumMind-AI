# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-07

### Added
- Initial project skeleton.
- Repository documentation (`CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`, `CHANGELOG.md`).
- Multi-persona Operations Dashboard frontend in Next.js.
- FastAPI backend with Pydantic validation and Gemini 2.5 Flash orchestration.
- 10 core stadium operating system features (Crowd AI, Emergency AI, Route Planner, Sustainability AI, etc.).
- CI/CD workflow under GitHub Actions.
- Unit testing setup for routes, models, and AI engine.
